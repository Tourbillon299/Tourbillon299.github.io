      python的高级特性
      一、切片

      L = [‘Michael’, ‘Sarah’, ‘Tracy’, ‘Bob’, ‘Jack’]
      笨方法就是
      [L[0], L[1], L[2]]
      取前n个就没办法了，就需要用到循环了
      切片就是可以这样
      L[0:3]
      L[:3]
      也可以倒数切片
      L[-2:]
      L[-2:-1]
      再次强调最后一个元素索引是-1
      L[:]
      原样复制一个list
      L[::2]是取奇数位元素

      也许对程序员来说很惊讶，学数学的表示，就那样吧。。

      二、迭代
      如果给定一个list或tuple，我们可以通过for循环来遍历这个list或tuple，这种遍历我们称为迭代

      d = {‘a’: 1, ‘b’: 2, ‘c’: 3}
      for key in d:
      … print(key)
      …
      a
      c
      b
      for ch in ‘ABC’:
      … print(ch)
      …
      A
      B
      C
      默认情况下，dict迭代的是key。如果要迭代value，可以用for value in d.values()，如果要同时迭代key和value，可以用for k, v in d.items()。
      判定是否可迭代，通过collections模块的Iterable类型判断（感觉用不到）。

      三、列表生成式
      比如，生成一个list[11,22,…,10*10]

      [x x for x in range(1, 11)]
      [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
      写列表生成式中，吧要生成的元素放到前面，后面跟for做循环。
      还可以跟上if判断，进一步筛选，如
      [x       x for x in range(1, 11) if x % 2 == 0]
      [4, 16, 36, 64, 100]
      两层循环可用于全排列，二维数组？？
      [m + n for m in ‘ABC’ for n in ‘XYZ’]
      [‘AX’, ‘AY’, ‘AZ’, ‘BX’, ‘BY’, ‘BZ’, ‘CX’, ‘CY’, ‘CZ’]

      四、生成器
      生成器：generator，用来一边循环一边计算的机制。
      创建办法一：把一个列表生成式的[]改成()。
      一个一个打印返回值可以用next()
      将返回值全部调用要使用for循环，因为generator也是可迭代对象
      创建办法二：如果一个函数定义中包含yield关键字，那么这个函数就不再是一个普通函数，而是一个generator。

      def fib(max):
      n, a, b = 0, 0, 1
      while n < max:
      print(b)
      a, b = b, a + b
      n = n + 1
      return ‘done’

      fib(6)
      1
      1
      2
      3
      5
      8
      ‘done’

      def fib(max):
      n, a, b = 0, 0, 1
      while n < max:
      yield b
      a, b = b, a + b
      n = n + 1
      return ‘done’

      f = fib(6)
      f

      ##返回的是对象

      for n in fib(6):
      … print(n)
      …
      1
      1
      2
      3
      5
      8

      五、迭代器
      可以直接作用于for循环的对象统称为可迭代对象，使用isinstance()判断一个对象是否是可迭代对象，可以被next()函数调用并不断返回下一个值的对象称为迭代器。
      生成器都是Iterator对象，但是list，dict，str虽然都是可迭代对象，但是不是迭代器，可以用iter()函数转换成迭代器。
      Python的for循环本质上就是通过不断调用next()函数实现的。
