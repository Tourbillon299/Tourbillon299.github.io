      四、条件判断
      也是用if，同样也有if else，但是有一个elif，是else if的缩写，形如
      if <条件判断1>
      <执行1>
      elif <条件判断2>
      <执行2>
      elif <条件判断3>
      <执行3>
      else:
      <执行4>

      在这里补充之前关于input的一点，input返回的数据类型为str，不能和整数直接比较，故，会出现这种情况
      birth = input(‘birth: ‘)
      if birth < 2000:
      print(‘00前’)
      else:
      print(‘00后’)
      然后报错
      方法是改为birth = int（input(‘birth: ‘)）

      练习

      小明身高1.75，体重80.5kg。请根据BMI公式（体重除以身高的平方）帮小明计算他的BMI指数，并根据BMI指数：

      低于18.5：过轻
      18.5-25：正常
      25-28：过重
      28-32：肥胖
      高于32：严重肥胖

      -- coding: utf-8 --

      height = 1.75
      weight = 80.5
      bmi = weight / height ** 2
      if bmi < 18.5:
      print(‘你的bmi指数为：%.1f，判定为过轻’ % bmi)
      elif bmi < 25:
      print(‘你的bmi指数为：%.1f，判定为正常’ % bmi)
      elif bmi < 28:
      print(‘你的bmi指数为：%.1f，判定为过重’ % bmi)
      elif bmi < 32:
      print(‘你的bmi指数为：%.1f，判定为肥胖’ % bmi)
      else:
      print(‘你的bmi指数为：%.1f，判定为严重肥胖’ % bmi)

      你的bmi指数为：26.3，判定为过重

      五、循环
      python的循环大概分两种，一种是for in循环，可以一次把list或者tuple里的每个元素拖出来
      a=[‘q’,’w’,’e’,’r’]
      for b in a:
      print(b)
      q
      w
      e
      r
      此处注意，python是一个对缩进要求非常严格的语言，该用teb和空格缩进的时候千万要缩进，否则会出错。

      range(i)函数是生成的序列是从0开始小于i的整数
      可以用list()转换成为list
      例如我想求1加到100的结果，就可以写成这样
      sum=0
      for x in range(101)：
      sum=sum+x
      print(sum)
      第二种是while循环，和c语言中相同。

      练习

      请利用循环依次对list中的每个名字打印出Hello, xxx!：

      #!/usr/bin/env python3

      -- coding: utf-8 --

      #用for循环来打印
      L = [‘B’, ‘L’, ‘A’]
      for name in L:
      print(‘Hello,%s!’%name)
      print(‘end\n’)

      #用while循环来打印
      n=3
      i=0
      while n>0:
      print(‘Hello,%s!’%L[i])
      n=n-1
      i=i+1
      print(‘end’)

      六、使用dict和set
      dict
      在其他语言里也有叫做map（你这样说我也不明白，还要去查map是什么）
      使用的是键-值（key-value）存储，具有极快的查找速度，也就是词典查法，有索引，有目录，查什么都不慢，一万个数据都一样快。

      d = {‘M’: 95, ‘B’: 75, ‘T’: 85}
      d[‘M’]
      95
      可以直接通过key赋值
      d[‘A’]=60
      多次赋值，后面的会把前面的覆盖掉。
      可以通过in和get判断list中存不存在想要找的key
      ‘T’in d
      False
      d.get(‘T’)
      d.get(‘T’, -1)
      -1
      删除操作也是使用pop()
      dict的缺点就是占用内存过大，因为采用计算key位置的算法是哈希算法，所以要保证每次计算相同的key得到的值相同，即key值必须是不可变对象。

      set
      和dict类似，也是key的一个集合，但是不存储value，key不能重复，重复的元素会自动过滤。

      s = set([1, 1, 2, 2, 3, 3])
      s
      {1, 2, 3}
      add(key)添加元素，remove(key)删除元素。
      set支持做数学上的交集，并集，差，标准差（并集-交集）等操作，符号是&，|，-，^
