      Post on 2016-07-01

      今天看两章基础，参考https://www.coursera.org/learn/hipython/
      python里加注释的方法是#，以#开头的语句是注释，注释是给人看的，可以是任意内容，解释器会忽略掉注释。其他每一行都是一个语句，当语句以冒号:结尾时，缩进的语句视为代码块。一般缩进都是四个空格，大小写敏感。

      一、数据类型和变量
      关于整数，没有什么需要特别注意的，和c基本相同。
      在浮点数中，科学计数法表示的是1.23e5也就是1.23*10的5次幂，也可以是12.3e4，整数运算永远是精确的，而浮点数运算则可能会有四舍五入的误差。
      字符串可以用双引号和单引号括起来的任何文本，如果字符串里有‘和”使用转义字符\表示，例如
      ‘I\’m \”JOJO\”‘
      I’m “JOJO”
      还有转义字符本身\用\表示，\n表示换行，\t表示制表符。
      布尔值使用True、False表示布尔值，可以使用and、or和not运算。与或非，等同。

      python特有一个值叫做空值，用None表示。不是0，只是空。
      python里的π是大写的PI

      关于除法运算

      10 / 3
      3.3333333333333335
      除法计算结果是浮点数，即使是两个整数恰好整除，结果也是浮点数
      9 / 3
      3.0
      还有一种整除，用//
      10 // 3
      3
      取余，用%
      10 % 3
      1

      练习

      打印以下变量的值：
      n = 123
      f = 456.789
      s1 = ‘Hello, world’
      s2 = ‘Hello, \’Adam\’’
      s3 = r’Hello, “Bart”‘
      s4 = r’’’Hello,
      Lisa!’’’
      print(‘n=123’)
      print(‘f=456.789’)
      print(“s1=’hello,world’”)
      print(“s2=’hello,\’adam\’’”)
      print(“s3=r\’hello,\”bart\”\’”)
      print(“s4=r\’\’\’hello,\nlisa!\’\’\’”)

      二、字符串编码

      字符串
      关于字符串，现在的Python 3版本中，字符串是以Unicode编码的，也就是支持多种语言的，
      可以用ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符，如

      ord(‘A’)
      65
      ord(‘乔’)
      20052
      chr（20052）
      ‘乔’
      用len（）可以获取字符串的长度
      len（’123’）
      3
      len()函数计算的是str的字符数，如果换成bytes，len()函数就计算字节数：

      len(b’ABC’)
      3
      len(b’\xe4\xb8\xad\xe6\x96\x87’)
      6
      len(‘中文’.encode(‘utf-8’))
      6

      %的用法
      %运算符就是用来格式化字符串的，%s表示用字符串替换，%d表示用整数替换，%f是浮点数，%x是十六进制整数，例如

      ‘Hello, %s’ % ‘world’
      ‘Hello, world’
      ‘Hi, %s, you have $%d.’ % (‘JOJO’, 1000000)
      ‘Hi, JOJO, you have $1000000.’
      其中格式化整数和浮点数还可以指定是否补0和整数与小数的位数，例如
      ‘%2d-%02d’ % (3, 1)
      ‘ 3-01’
      ‘%.2f’ % 3.1415926
      ‘3.14’
      %%转义的就是%字符自己。

      练习

      小明的成绩从去年的72分提升到了今年的85分，请计算小明成绩提升的百分点，并用字符串格式化显示出’xx.x%’，只保留小数点后1位：

      #!/usr/bin/env python3

      -- coding: utf-8 --

      s1 = 72
      s2 = 85
      r = (s2-s1)*100/s1

      print(‘\’%.1f%%\’’ % r)

      三、关于python里的数据结构
      list
      list是一个有序的集合，可以随时添加和删除其中的元素。

      a=[‘a’,’b’,’c’]
      [‘a’,’b’,’c’]
      最后一个元素的索引是len（a）-1，选取最后一个元素可以直接用-1做索引，
      a[-1]
      ‘c’
      同理-2，-3就是倒数第二个第三个。，注意不要越界。
      list里可以往最后添加元素，也可以插入到指定位置，分别是append和insert，例如
      classmates = [‘Michael’, ‘Bob’, ‘Tracy’]
      classmates.append(‘Adam’)
      classmates
      [‘Michael’, ‘Bob’, ‘Tracy’, ‘Adam’]
      classmates.insert(1, ‘Jack’)
      classmates
      [‘Michael’, ‘Jack’, ‘Bob’, ‘Tracy’, ‘Adam’]
      删除list末尾的元素要用pop()
      classmates.pop()
      ‘Adam’
      classmates
      [‘Michael’, ‘Jack’, ‘Bob’, ‘Tracy’]
      删除指定位置就给pop加参数pop（i）i是索引值
      classmates.pop(1)
      ‘Jack’
      classmates
      [‘Michael’, ‘Bob’, ‘Tracy’]
      要把某个元素替换成别的元素，可以直接赋值给对应的索引位置：

      classmates[1] = ‘Sarah’
      classmates
      [‘Michael’, ‘Sarah’, ‘Tracy’]

      list里面的元素的数据类型也可以不同，比如：

      L = [‘Apple’, 123, True]

      list元素也可以是另一个list，比如：

      s = [‘python’, ‘java’, [‘asp’, ‘php’], ‘scheme’]
      len(s)
      4
      空list的长度为0.

      tuple
      这个有序列表叫元组，和list不同的是初始化之后不能修改

      t = (1, 2)
      t
      (1, 2)
      但是因为括号有多种用法，在表示单个元素的元组时，必须加一个逗号，如
      t = (1,)
      t
      (1,)

      练习
      请用索引取出下面list的任意三个元素：

      -- coding: utf-8 --

      L = [
      [‘Apple’, ‘Google’, ‘Microsoft’],
      [‘Java’, ‘Python’, ‘Ruby’, ‘PHP’],
      [‘Adam’, ‘Bart’, ‘Lisa’]
      ]

      打印Apple:

      print(L[0][0])

      打印Python:

      print(L[1][1])

      打印Lisa:

      print(L[2][2])
